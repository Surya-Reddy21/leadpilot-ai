# LeadPilot AI

Turn website visitors into leads with an AI support agent that answers questions, books appointments, and escalates to humans.

## Running the project

1. Install dependencies:
   - `npm install`
2. Start the backend and frontend together:
   - `npm run dev`
3. Use the frontend at `http://localhost:3000` and the API at `http://localhost:5000`

## MVP features

- Customer chat assistant
- FAQ management
- Lead capture
- Appointment booking
- Analytics dashboard
- Human escalation tracking
